SYNOPSIS

Script para llevar una lista de cosas para hacer.

Permite agregar tareas, agruparlas en categorias, asignar y re-asignar prioridades, borrar por ID o categorias completas y exportar todo a HTML.

Sin opciones despliega una lista simple de cosas para hacer.

Forma de uso:

PA_HACER -t Hola -c Prueba -p 1

PA_HACER

PA_HACER -X

PA_HACER -B Prueba

PA_HACER -p 9 -i 1

Opciones

-t Tarea - Descripcion o titulo de la tarea en si.
-c Categoria - Las categorias agrupan tareas. [ prescindible ]
-p Prioridad - La prioridad debe ser numerica y entre 1-10. [ prescindible ]
-B Borrar tarea/s - Permite eliminar tareas de la lista, por ID o CATEGORIA. [ excluyente ]
-X Exportar la lista de tareas a HTML. [ excluyente ]
-p Priorizar : Toma un numero que sera asignado como nueva prioridad para la tarea con ID -i. [ Necesita -i]
-i ID de la tarea a priorizar. [ Necesita -p]
-n NUEVO archivo TODO.txt: Borra el viejo, cuidadito... [excluyente]
-a ARCHIVAR - Backup del archivo TODO.txt actual. [excluyente]
-r RESTAURAR - Permite restaurar algun backup y convertirlo en acutal. [excluyente]
-d DEBUGGING FLAG!
-h (Esta) Ayuda. [excluyente]

Los parametros presindibles toman valores por defecto ante su ausencia.

Las opciones excluyentes NO se pueden combinar con otras.

Las opciones -p y -i van siempre juntas.

La prioridad por defecto es de 6.

La categoria por defecto es Etc..

Ejemplos:

Para insertar una tarea, alcanza con utilizar solamente la opcion -t.

PA_HACER -t Hola -c Prueba -p 1

La opciones -c y -p no son obligatorias y asignan prioridad y categoria.

La tarea es -en si misma- una descripcion, y conviene agruparlas en categorias para borrarlas mas facilmente.

Las prioridades, a su vez, se pueden reasignar.

TXT

Todo lo que este programita hace esta en un archivo de texto plano (el viejo y querido txt).
